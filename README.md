## Supported Browers
* Google Chrome v39+ - developed on Google Chrome v39+ and is deemed to work
* FireFox 35.0.1+ - tested on FireFox for Mac, it works.

## Unsupported Browers
* Mobile Browsers: Although it may play, the sizing is not optimised for it.
